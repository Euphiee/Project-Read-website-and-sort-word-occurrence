课题 - 读取网页，统计每个字出现次数并排序
语言：Python
python库：requests(读取网页), BeautifulSoup(处理html标签), re(处理文本), time(计算运行时间), os(计算运行内存), psutil(计算运行内存)
运行：
1. 运行final_code.py
2. 输入要读取的网页url
3. 输入要统计的语言（英语，中文，日文）
4. 结果将导出在data.csv文件中


Project - Sort words in a website according to frequency of occurrence
Language: python
Library used: requests(read website), BeautifulSoup(process html tags), re(process text), time(calculate running time), os(calculate running memory), psutil(calculate running memory)
Instruction:
1. run final_code.py in terminal
2. enter a website url
3. enter the result language (English, Japanese or Chinese)
4. result will be exported and saved to data.csv file in the same directory.
